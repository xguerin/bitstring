How to install

opam install bitstring
opam install ppx_bitstring

How to use

Ocamlfind

ocamlfind c -package bitstring -package ppx_bitstring -linkpkg ...

Dune

(executable
 ((name        foo)
  (libraries   (bitstring))
  (preprocess  (pps ppx_bitstring))
  ))

How to build

Building the project

$ dune build

Running the tests

$ dune runtest
